# SSH Manager

Интерактивное TUI-меню для управления SSH на **Ubuntu 24.04**.  
Без GUI — только терминал, цифры, подтверждения.

---

## Установка одной командой

```bash
curl -fsSL https://raw.githubusercontent.com/Lordtop4ik/ssh_manager/refs/heads/main/install.sh | sudo bash
```

---

## Запуск

```bash
sudo ssh_manager
```

---

## Возможности

```
==== SSH MANAGER ====

1. Управление ключами
2. Пользователи
3. Настройки SSH
4. Безопасность
5. Откат
6. Мастер безопасной настройки   ← рекомендуется
0. Выход
```

### 1. Управление ключами
- Добавить ключ (вручную или из файла) с проверкой формата
- Удалить ключ (выбор из списка)
- Показать все ключи пользователя

### 2. Пользователи
- Создать пользователя (имя, пароль, SSH-ключ, sudo)
- Удалить пользователя
- Выдать / убрать sudo

### 3. Настройки SSH
- Показать текущие параметры sshd_config
- Изменить порт (с проверкой занятости)
- Перезапустить SSH (с валидацией конфига)

### 4. Безопасность
- Запретить / разрешить вход root
- Отключить / включить вход по паролю
- Защита: предупреждение если нет SSH-ключей

### 5. Откат
- Восстановить sshd_config из резервной копии
- Аварийный сброс (включить пароль и root)

### 6. Мастер безопасной настройки ⭐
Пошаговый мастер для безопасного первичного конфигурирования:
1. Создаёт нового пользователя + SSH-ключ + sudo
2. (Опционально) Меняет порт
3. Перезапускает SSH
4. **Даёт время проверить доступ из нового окна**
5. Только после подтверждения — отключает root и пароль

---

## Безопасность

- Все опасные действия требуют подтверждения `(y/n)`
- Перед каждым изменением создаётся резервная копия `/etc/ssh/sshd_config.bak`
- Перед рестартом выполняется `sshd -t` (валидация конфига)
- При отключении пароля без SSH-ключей — предупреждение и блокировка
